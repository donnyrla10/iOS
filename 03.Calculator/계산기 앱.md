<br>

### **💙 기능**

- 숫자판을 누르면 원하는 값을 입력할 수 있다.
- `+, -, *, /` 를 누르면 해당 연산자로 값을 계산 할 수 있다.
- `AC`를 누르면 값을 초기화한다.
- `+/-` 를 누르면 부호 변환
- `%` 를 누르면 퍼센트 계산

<br>

### **💙 강의 외의 추가적인 구현**
- `AC ↔ C` 변환
- `+/-` 기능 구현 (부호 변환)
- `%` 기능 구현 (퍼센트 연산) ✅ (*TEST 추가적으로 필요*)
- `UI` 변환 ✅
- 버튼 클릭시 버튼 색상 변경
