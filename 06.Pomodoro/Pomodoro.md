# Pomodoro ⏲

<br>


뽀모도로 앱 ⏲

- DatePicker를 통해 타이머 시간 설정
- 시작 버튼 → 타이머 시작
- 일시정지 버튼 → 타이머 일시정지
- 취소 버튼 → 타이머 종료
- 카운트 다운 완료 → 알람 울림  



➿ 뽀모도로는 시간 관리 방법론으로, 타이머를 이용해서 25분간 집중해서 일을 한 다음 5분간 휴식을 취하는 방식이다. 
현재 구현한 것은 타이머 기능만 있으므로 추후에 25분 집중 - 5분 휴식 할 수 있는 기능을 추가할 것이다
